About This Project

This site was created as a learning companion for the Yahboom Building:bit Super Kit - a fantastic robotics kit that combines LEGO-compatible building blocks with the BBC micro:bit.

The original Yahboom documentation site has all the content but can be difficult to navigate. This project reorganizes the 32 builds into an easy-to-browse gallery with step-by-step assembly instructions and MakeCode/Python programming lessons.

Built with love by a dad helping his son explore robotics and programming.
